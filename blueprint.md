# Lotto Number Generator

## Overview

This application generates a set of 6 unique random numbers for a lottery drawing. It provides a simple and clean interface for the user to generate and view the numbers.

## Features

*   Generates 6 unique random numbers between 1 and 45.
*   Displays the generated numbers in a visually appealing way.
*   A button to generate a new set of numbers.
*   Modern and responsive design using modern CSS.
*   Uses a Web Component to display the numbers.
*   Enhanced visual design with custom fonts, vibrant colors, and subtle animations.
